## 🐛 常见问题排查

### Q1: 加血包还是收集不到？
**排查步骤**：
1. 打开Console（F12）
2. 飞向加血包
3. 观察是否有任何错误信息
4. 检查距离是否足够近（<50像素）
5. 截图Console内容反馈

### Q2: 看不到其他玩家？
**解决方案**：
1. 确保两个设备使用相同的网址
2. 检查网络连接
3. 等待几秒钟让数据同步
4. 刷新页面重试

### Q3: 碰撞没有推开效果？
**排查步骤**：
1. 确认能看到对方的气球
2. 确保距离足够近（<80像素）
3. 检查Console是否有碰撞日志
4. 尝试直接撞过去，不要绕圈

### Q4: 修改配置后没生效？
**解决方案**：
1. 确认点击了"保存参数"
2. 等待下一次元素生成（不是立即）
3. 清除浏览器缓存后刷新
4. 检查Console是否有错误
