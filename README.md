# 🎈 多人实时热气球飞行游戏

一个基于 Phaser.js 和 Supabase 的多人实时互动游戏。

## ✨ 特性

- 🎨 像素风热气球（红白条纹 + 吊篮）
- ☁️ 动态流云背景
- 🎮 点击飞行，平滑移动
- 👥 多人实时同步
- 💫 金闪闪广告位

## 🎮 如何玩

1. 点击屏幕任意位置
2. 气球会平滑飞向目标
3. 打开多个标签页或分享给朋友，体验多人实时同步
